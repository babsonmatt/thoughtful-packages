# thoughtful-packages


Install Bun
```bash
curl -fsSL https://bun.sh/install | bash
```

Clone the repository
```bash
git clone https://github.com/babsonmatt/thoughtful-packages
```

Change into the directory
```bash
cd thoughtful-packages
```

Install the dependencies
```bash
bun install
```

Run the tests
```bash
bun test
```
